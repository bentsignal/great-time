<h2 align="center">Great Time</h2>
<p align="center">Simple unit conversion in TypeScript</p>

### Overview

Convert time units in TypeScript

### Installation

```
ni great-time
```

```
pnpm i great-time
```

```
bun i great-time
```

```
npm i great-time
```

```
yarn add great-time
```

### Usage

```typescript
import { convert } from "great-time";

const time = convert(2, "weeks", "to ms");
```
