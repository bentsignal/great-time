<h2 align="center">Great Time</h2>
<p align="center">Simple unit conversion in TypeScript</p>

### Overview

Simple time unit conversion for typescript.

### Installation

```
ni ts-package
```

```
pnpm i ts-package
```

```
bun i ts-package
```

```
npm i ts-package
```

```
yarn add ts-package
```

### Usage

```typescript
import { convert } from "great-time";

const time = convert(2, "weeks", "to ms");
```
